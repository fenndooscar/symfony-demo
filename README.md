# Symfony API Demo

Простое Demo API приложение с использованием фреймворка Symfony.

Запуск:
```shell
sh runner.sh start
```

Анализ кода (phpstan):
```shell
sh runner.sh cs:analyse
```

Код стайл фиксер (php-cs-fixer):
```shell
sh runner.sh cs:fix
```
